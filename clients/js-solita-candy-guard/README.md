# tpl-candy-guard

This package contains the Trezoaplex Candy Guard contract SDK code.

## API Docs

Find the [candy-guard API docs published here](https://trezoaplex-foundation.github.io/tpl-candy-guard/index.html).

## Installation

```shell
npm install @trezoaplex-foundation/tpl-candy-guard --save
```

## Developing

In order to update the generated SDK when the rust contract was updated please run:

```
yarn api:gen
```

and then update the wrapper code and tests.
